# Track: Enhanced Search with Regex Support

## Overview
Implement an advanced filtering tool for downloads using regular expressions on filenames, URIs, and tracker URLs.

## Functional Requirements
- **Regex Filter**: Allow users to specify a regex pattern for download metadata.
- **Support for Multi-Metadata Search**: Searching by filename, URI, and tracker URL.
- **In-Memory Filtering**: Fast, in-memory filtering of active, waiting, and stopped downloads.

## Acceptance Criteria
- [ ] Users can provide a regex pattern to the search tool.
- [ ] The tool correctly filters and returns downloads matching the pattern.
- [ ] Multiple metadata fields can be used for searching.
