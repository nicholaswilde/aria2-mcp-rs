# Specification: Increase Code Coverage

## Goal
Increase the total code coverage of the project to at least 90%.

## Target Areas
1.  **server/stdio.rs** (Current: 58.54%): Improve coverage for the stdio server loop and error handling.
2.  **state.rs** (Current: 67.44%): Improve coverage for state management, persistence, and error cases.
3.  **main.rs** (Current: 83.45%): Test CLI argument parsing and error initialization more thoroughly.
4.  **server/sse.rs** (Current: 82.60%): Improve coverage for SSE stream handling and authentication middleware.

## Success Criteria
- Total line coverage >= 90%.
- No regression in existing tests.
